Fare stage service shortlist:

MODEL 

* FSService allows edit sequence of Fare Stages, 
  collection of Bus Stops assigned to Fare Stage and 
  Direction on Bus Stop in Fare Stage.

* Fare Stage and Bus Stop lists are constant and got outside the Fare Stage Service editor. 
* Fare Stage can occure multiple times in FSService collection, 
  but only one instance must be used.
* Bus Stop can occure in multiple Fare Stages. 
  Bus Stop can occure multiple times in the one Fare Service, but as one instance. 
  Direction of Bus Stop must be set per Fare Stage.
